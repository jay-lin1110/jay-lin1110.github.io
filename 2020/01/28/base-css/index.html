封装基本公共样式工具类库

CSS SASS
SASS
发布日期:   2020-01-28
文章字数:   1.3k
阅读时长:   5 分
阅读次数:  

使用 SASS 自定义封装一个类似于 Bootstrap 风格的基本可复用样式工具库，将常见的基本样式封装到一个统一的 base.css 文件中，提供对应的基本工具样式类

基本公共样式工具库

在项目开发中，我们提前封装好一个基本的公共可复用的工具样式类文件，然后在项目的入口文件引入一次，就可以在整个项目中使用这些基础的公共样式类，避免多次写同样的基本样式。

比如，在 vue-cli 的脚手架项目中，我们可以在入口文件引入封装好的一个公共样式类，通常我们还会在这里引入对应的重置样式文件，vue 中中我们有 webpack 相关的 loader 帮我们处理编译打包 SASS 文件，所以不需要额外的处理。

如果你的项目支持 SASS 语法，那你可以直接使用 SASS 编写封装，这将提供极大的便利和良好的开发体验。

如果你的项目不支持 SASS 语法，那你可以使用 SASS 编写，再使用 VSCode 等编辑器相应的插件提前编译好 CSS 文件，再在项目中使用。

同理，微信小程序原生 Mina 框架并不支持 SASS 语法编写，同时文件也是自己原生框架生态系统的 WXSS 文件，我们可以使用插件将 SASS 文件编译为 WXSS 文件，然后在项目中使用。

如果你对 SCSS 语法不熟悉，可以使用 SASS 官网进行学习，Sass 是世界上最成熟、稳定、强大的专业级 CSS 扩展语言，可以帮助我们提供良好的样式开发体验。

基础样式工具类

字体

以字体样式为例子，字体样式通常包括字体的对齐方式、字体颜色、字体大小等基本样式，所以我们可以对这部分样式进行封装。这里，我们将使用 SCSS 语法，包括但不限于变量、循环、插值和集合等。

我们可以先定义部分主题相关或者常用的颜色变量。

$theme: #d63246;
$white: #fff;
$black: #000;
$dark: #333;
$gray: #ccc;
$gray-1: #eee;

$colors: (
  theme: $theme,
  white: $white,
  black: $black,
  dark: $dark,
  gray: $gray,
  gray-1: $gray-1
);

然后我们借助 SASS 语法，通过循环和插值编译生成我们需要的字体样式工具类

@each $colorKey, $color in $colors {
  .text-#{$colorKey} {
    color: $color;
  }
}

上面的 SASS 代码定义了我们字体颜色相关的样式工具类，编译后生成的 CSS 样式文件如下：

.text-theme {
  color: #d63246;
}

.text-white {
  color: #fff;
}

.text-black {
  color: #000;
}

.text-dark {
  color: #333;
}

.text-gray {
  color: #ccc;
}

.text-gray-1 {
  color: #eee;
}

同理，我们可以生成背景颜色相关的文件。

@each $colorKey, $color in $colors {
  // text color
  .text-#{$colorKey} {
    color: $color;
  }
  // background color
  .bgc-#{$colorKey} {
    background-color: $color;
  }
}

编译后样式文件如下：

.text-theme {
  color: #d63246;
}

.bgc-theme {
  background-color: #d63246;
}

.text-white {
  color: #fff;
}

.bgc-white {
  background-color: #fff;
}

.text-black {
  color: #000;
}

.bgc-black {
  background-color: #000;
}

.text-dark {
  color: #333;
}

.bgc-dark {
  background-color: #333;
}

.text-gray {
  color: #ccc;
}

.bgc-gray {
  background-color: #ccc;
}

然后在我们的 html 文件中或组件中使用字体基本样式工具类。

<div class="bgc-gray">
  <p class="text-theme">Hello World!</p>
  <p class="text-dark">Hello World!</p>
</div>

我们可以定义更多字体相关的样式，比如字体的大小，字体的对齐方式等等。我们可以设置基础的字体大小，根据倍数计算出不同大小、不同级别的样式。

$align: (left, right, center);

@each $value in $align {
  // text align
  .text-#{$value} {
    text-align: $value;
  }
}
// 基本字体大小
$base-font-size: 1rem;
// 字体大小的倍数规格
$font-sizes: (
  xs: 0.5,
  sm: 0.75,
  md: 1,
  lg: 1.25,
  xl: 1.5
);

@each $sizeKey, $size in $font-sizes {
  // font size
  .fs-#{$sizeKey} {
    font-size: $size * $base-font-size;
  }
}

flex

我们可以定义 flex 布局相关的样式，包括方向、布局方式、主轴排列方式、交叉轴排列方式等。

// flex布局
.d-flex {
  display: flex;
}
// flex主轴方向
.flex-column {
  flex-direction: column;
}

$jc: (
  start: flex-start,
  end: flex-end,
  center: center,
  between: space-between,
  around: space-around
);

@each $key, $value in $jc {
  .jc-#{$key} {
    justify-content: $value;
  }
}

$ai: (
  start: flex-start,
  end: flex-end,
  center: center,
  stretch: stretch
);

@each $key, $value in $jc {
  .ai-#{$key} {
    align-items: $value;
  }
}

边距

边距样式与字体样式相似，我们编译生成不同大小、不同级别、水平方向或垂直方向等的样式。比如我们生成的样式名字为 mt-1 ，对应的样式就是 margin-top 一级大小的边距样式，我们可以自由设置类名。

// 边距类型
$spacing-types: (
  m: margin,
  p: padding
);
// 边距方向
$spacing-directions: (
  t: top,
  b: bottom,
  l: left,
  r: right
);
// 基本边距大小
$spacing-base-size: 1rem;
// 边距的倍数等级
$spacing-sizes: (
  0: 0,
  1: 0.25,
  2: 0.5,
  3: 1.75,
  4: 1.5,
  5: 2
);

@each $typeKey, $type in $spacing-types {
  @each $directionKey, $direction in $spacing-directions {
    @each $sizeKey, $size in $spacing-sizes {
      // m-1
      .#{$typeKey}-#{$sizeKey} {
        #{$type}: $spacing-base-size * $size;
      }
      // mx-1
      .#{$typeKey}x-#{$sizeKey} {
        #{$type}-left: $spacing-base-size * $size;
        #{$type}-right: $spacing-base-size * $size;
      }
      // my-1
      .#{$typeKey}y-#{$sizeKey} {
        #{$type}-top: $spacing-base-size * $size;
        #{$type}-bottom: $spacing-base-size * $size;
      }
      // mt-1
      .#{$typeKey}#{$directionKey}-#{$sizeKey} {
        #{$type}-#{$direction}: $spacing-base-size * $size;
      }
    }
  }
}
